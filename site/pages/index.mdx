## Search, Filter, Pipe

</div>

```bash
# Basic one-way search
crosswind BEG JFK apr1

# Round-trip, business class
crosswind BEG JFK apr1 -r apr16 -c business

# Nonstop only, JSON output
crosswind LAX JFK tomorrow --max-stops 0 --json

# Compare multiple destinations
crosswind BEG JFK,LHR,CDG apr1 | jq '.data.flights[:3]'

# Open in browser instead
crosswind BEG JFK apr1 --open
```

## How It Works

</div>

<div style={{ fontSize: '15px', opacity: 0.8, lineHeight: 1.7, margin: '16px 0 48px' }}>
Crosswind encodes your search as a Protocol Buffer, fetches the Google Flights page with Chrome TLS fingerprinting, and extracts structured flight data from the embedded JSON payload. No API key needed. No headless browser. Single static binary.
